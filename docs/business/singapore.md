# Singapore Education Market Entry Guide

## Overview

This document outlines the pathways for Shenbi to enter the Singapore education market, including government procurement processes, certifications, and direct school partnerships.

---

## Market Context

### Singapore Education System

| Level | Age | Schools | Notes |
|-------|-----|---------|-------|
| Primary | 7-12 | ~180 schools | Compulsory education |
| Secondary | 13-16 | ~150 schools | Various tracks (Express, Normal, IP) |
| International | 3-18 | ~70 schools | Higher budget, flexible curriculum |

### EdTech Landscape

- **MOE EdTech Masterplan 2030**: Government's strategic plan for educational technology
- **Singapore Student Learning Space (SLS)**: MOE's primary digital learning platform
- **High Digital Adoption**: 1:1 device programs in most schools
- **Strong Government Support**: Significant budget for EdTech initiatives

## Entry Pathways

### Path 1: IMDA Accreditation (Recommended)

The [IMDA Accreditation Programme](https://www.imda.gov.sg/how-we-can-help/imda-accreditation) is designed for Singapore-based tech companies seeking government contracts.

#### Benefits

| Benefit | Description |
|---------|-------------|
| Priority Consideration | Government agencies consider accredited companies first |
| Decision Maker Access | Regular workshops and showcases with government buyers |
| Publicity Support | Help with branding and visibility |
| Fundraising Assistance | Support in investor connections |
| Track Record | $814M+ in government contracts awarded to accredited companies |

#### Eligibility Requirements

```
┌─────────────────────────────────────────────────────────────┐
│                   IMDA Accreditation Criteria               │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ✓ Singapore-registered company (ACRA)                      │
│  ✓ Substantial business presence in Singapore               │
│  ✓ Annual group revenue < $100 million                      │
│  ✓ Own the product intellectual property                    │
│  ✓ Not a reseller/distributor                               │
│  ✓ Innovative product with unique value proposition         │
│  ✓ Local R&D, testing, and technical support                │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### Evaluation Areas

| Area | What They Assess |
|------|------------------|
| Technical | Product functionality, architecture, security |
| Financial | Company financial health, sustainability |
| Operational | Delivery capability, support infrastructure |

#### Application Process

```
Step 1: Self-Assessment
   │     Review eligibility criteria
   │     Prepare documentation
   ↓
Step 2: Application Submission
   │     Submit via IMDA portal
   │     Pay application fee (if any)
   ↓
Step 3: Technical Evaluation
   │     Product demonstration
   │     Architecture review
   │     Security assessment
   ↓
Step 4: Financial & Operational Review
   │     Financial statements review
   │     Team capability assessment
   │     Support infrastructure check
   ↓
Step 5: Accreditation Decision
   │     Committee review
   │     Approval/feedback
   ↓
Step 6: Onboarding to AC Panel
         Listed as accredited company
         Access to government opportunities
```

**Timeline**: 3-6 months typically

### Path 2: GeBIZ Government Procurement

[GeBIZ](https://www.gebiz.gov.sg/) is Singapore Government's one-stop e-procurement portal.

#### How It Works

```
┌─────────────────────────────────────────────────────────────┐
│                    GeBIZ Procurement Flow                   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Government Agency (e.g., MOE)                              │
│         │                                                   │
│         ▼                                                   │
│  Posts Tender/Quotation on GeBIZ                           │
│         │                                                   │
│         ▼                                                   │
│  Suppliers Download Requirements                            │
│         │                                                   │
│         ▼                                                   │
│  Suppliers Submit Bids                                      │
│         │                                                   │
│         ▼                                                   │
│  Evaluation (Price + Quality)                               │
│         │                                                   │
│         ▼                                                   │
│  Award to Best Value Supplier                               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### Registration Requirements

| Entity Type | Requirements |
|-------------|--------------|
| ACRA-Registered | Register directly via GeBIZ portal |
| Non-ACRA (Foreign) | Additional verification process required |

#### Tender Types

| Type | Value | Process |
|------|-------|---------|
| Small Value Purchase | < $6,000 | Direct purchase |
| Invitation to Quote (ITQ) | $6,000 - $90,000 | Min 3 quotes |
| Invitation to Tender (ITT) | > $90,000 | Open tender |

#### Evaluation Criteria

Government evaluates based on "value for money":

- Suitability for purpose
- Quality and reliability
- Risk assessment
- Timeliness of delivery
- Total cost of ownership
- Economic, social, environmental factors

#### Tips for Success

1. **Monitor Regularly**: Check GeBIZ daily for new MOE opportunities
2. **Understand Requirements**: Read tender documents thoroughly
3. **Price Competitively**: Government seeks best value, not lowest price
4. **Demonstrate Capability**: Include case studies and references
5. **Meet All Criteria**: Non-compliant bids are rejected

### Path 4: Direct School Partnerships

Bypass government procurement by partnering directly with schools.

#### Opportunity Types

| Type | Description | Decision Maker |
|------|-------------|----------------|
| CCA (Co-Curricular Activity) | After-school coding club | HOD CCA |
| Enrichment Programme | Curriculum enrichment | HOD ICT / Principal |
| After-School Care | ASC center activities | ASC Operator |
| Parent-Funded | Optional paid program | Parents |
| International Schools | Flexible curriculum | Head of Technology |

#### Approach Strategy

```
┌─────────────────────────────────────────────────────────────┐
│              Direct School Partnership Process              │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Step 1: Identify Target Schools                            │
│          - Progressive schools known for innovation         │
│          - Schools with strong ICT programs                 │
│          - International schools (faster decision)          │
│                                                             │
│  Step 2: Initial Contact                                    │
│          - Email HOD ICT or VP (Admin)                      │
│          - Offer free trial/demo                            │
│          - Share case studies                               │
│                                                             │
│  Step 3: Demo & Pilot                                       │
│          - On-site demonstration                            │
│          - Free 1-month pilot with 1-2 classes              │
│          - Collect feedback and data                        │
│                                                             │
│  Step 4: Proposal & Negotiation                             │
│          - Present ROI and learning outcomes                │
│          - Flexible pricing for pilot schools               │
│          - Multi-year discount options                      │
│                                                             │
│  Step 5: Implementation                                     │
│          - Teacher training                                 │
│          - Ongoing support                                  │
│          - Regular progress reviews                         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Regulatory Considerations

### Data Protection (PDPA)

Singapore's Personal Data Protection Act applies to student data.

| Requirement | Action |
|-------------|--------|
| Consent | Obtain consent for data collection |
| Purpose Limitation | Use data only for stated purposes |
| Data Retention | Don't keep data longer than necessary |
| Data Security | Implement appropriate security measures |
| Access & Correction | Allow users to access/correct their data |

### Child Safety

| Area | Requirement |
|------|-------------|
| Content | Age-appropriate, no harmful content |
| Privacy | Minimize personal data collection |
| Advertising | No targeted advertising to children |
| Contact | Parental consent for any communication |

### Accessibility

- Support for students with special needs
- Compliance with WCAG 2.1 guidelines recommended
- Consider dyslexia-friendly fonts and color schemes

## Go-to-Market Timeline

### Phase 1: Foundation (Months 1-3)

- [ ] Register Singapore company (ACRA)
- [ ] Set up local bank account
- [ ] Register for GeBIZ
- [ ] Begin IMDA Accreditation application
- [ ] Identify 5 target pilot schools

### Phase 2: Pilot (Months 4-6)

- [ ] Conduct demos at target schools
- [ ] Launch 2-3 free pilots
- [ ] Collect testimonials and case studies
- [ ] Refine product based on feedback
- [ ] Build relationships with HODs

### Phase 3: Commercial Launch (Months 7-12)

- [ ] Convert pilots to paid contracts
- [ ] Receive IMDA Accreditation
- [ ] Respond to GeBIZ tenders
- [ ] Scale to 10+ schools
- [ ] Hire local sales/support team

### Phase 4: Scale (Year 2)

- [ ] Target MOE bulk contracts
- [ ] Expand to 50+ schools
- [ ] Launch B2B SaaS for enrichment centers
- [ ] Explore Malaysia/regional expansion
