# 中国市场运营指南

## 概述

本文档为中国创业者/公司运营少儿编程业务提供实操指南，包括资质办理、合规要求、运营策略等。

---

## 市场背景

### 市场规模

| 年份 | 市场规模 | 增长率 |
|------|---------|--------|
| 2022 | 324亿元 | 18% |
| 2023 | ~380亿元 | ~17% |
| 2025E | 515亿元 | CAGR 17% |

### 市场渗透率

- 当前渗透率：**不足 2%**
- 用户规模（2023）：约 **160万人**
- 主要集中区域：华东（>30%）、华南、华北

### 政策环境

#### 双减政策影响

2021年"双减"政策出台后：
- ❌ 学科类培训受严格限制
- ✅ 非学科类（含编程）被定位为"科学加法"
- ✅ 素质教育培训可参与学校课后服务
- ✅ 人工智能教育被纳入国家战略

#### 利好政策

| 政策 | 发布时间 | 影响 |
|------|---------|------|
| 《教育强国建设规划纲要（2024-2035）》 | 2024 | 强调科学教育、数字素养 |
| 《北京市中小学人工智能教育方案（2025-2027）》 | 2025 | AI教育进校园，引入社会力量 |
| 教育部加强中小学人工智能教育通知 | 2024 | 全国性推广AI/编程教育 |

## 资质要求

### 一、公司注册（内资企业）

国内公司注册流程非常简单：

```
┌─────────────────────────────────────────────────────────────┐
│                    内资公司注册流程                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Step 1: 公司名称核准（1-3天）                               │
│     ↓    市场监督管理局网上系统                              │
│                                                             │
│  Step 2: 提交设立申请（1-3天）                               │
│     ↓    可全程网上办理                                      │
│                                                             │
│  Step 3: 领取营业执照（当天）                                │
│     ↓    市场监督管理局                                      │
│                                                             │
│  Step 4: 刻章（当天）                                        │
│     ↓    公章、财务章、法人章                                │
│                                                             │
│  Step 5: 银行开户（3-7天）                                   │
│     ↓    任选一家银行                                        │
│                                                             │
│  Step 6: 税务登记（当天）                                    │
│          税务局或网上办理                                    │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**总周期**：约 1-2 周
**费用**：几百到几千元（含代办费）

#### 经营范围建议

```
教育咨询服务；教育软件开发；计算机技术培训；
组织文化艺术交流活动；教学设备销售；
信息技术咨询服务；软件开发；技术服务；
从事互联网文化活动（如需线上业务）
```

**注意**：如果只做纯线上SaaS平台，不直接招生授课，可能不需要办学许可证。

### 二、办学许可证（线下培训必需）

#### 是否需要办学许可证？

| 业务模式 | 是否需要 | 说明 |
|---------|---------|------|
| 线下培训机构（直接招生） | ✅ 必须 | 非学科类培训许可 |
| 纯SaaS平台（卖软件给机构） | ❌ 不需要 | 你不是培训主体 |
| 线上录播课销售 | ⚠️ 看情况 | 部分地区有要求 |
| 进校提供课程服务 | ✅ 必须 | 需进"白名单" |

#### 非学科类培训机构设立要求

| 要求项 | 具体内容 |
|-------|---------|
| **场地要求** | 符合消防、环保、卫生标准；不能在居民楼 |
| **面积要求** | 通常要求 200㎡ 以上（各地不同） |
| **楼层要求** | 培训用房不超过5层（部分地区4层） |
| **资金要求** | 注册资本/开办资金 10-30万（各地不同） |
| **师资要求** | 持证教师，无犯罪记录 |
| **课程要求** | 培训目标、计划、内容需备案 |

#### 办学许可证申请流程

```
┌─────────────────────────────────────────────────────────────┐
│                  办学许可证申请流程                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Step 1: 选址并签租赁合同                                    │
│     │    确保场地符合要求（面积、楼层、消防）                 │
│     ↓                                                       │
│                                                             │
│  Step 2: 消防验收/备案                                       │
│     │    联系消防部门进行验收                                │
│     │    面积>300㎡需验收，<300㎡可备案                       │
│     ↓                                                       │
│                                                             │
│  Step 3: 准备申请材料                                        │
│     │    - 场地证明/租赁合同                                 │
│     │    - 消防验收合格证明                                  │
│     │    - 师资资质证明                                      │
│     │    - 课程大纲及教材                                    │
│     │    - 管理制度文件                                      │
│     │    - 资金证明                                          │
│     ↓                                                       │
│                                                             │
│  Step 4: 提交申请                                           │
│     │    向当地教育局或行政审批局提交                        │
│     ↓                                                       │
│                                                             │
│  Step 5: 现场核查                                           │
│     │    教育、消防等部门实地检查                            │
│     ↓                                                       │
│                                                             │
│  Step 6: 颁发许可证                                         │
│     │    教育行政部门颁发办学许可证                          │
│     ↓                                                       │
│                                                             │
│  Step 7: 法人登记                                           │
│          市场监管局（营利性）或民政局（非营利）              │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**周期**：通常 2-4 个月（各地效率不同）

**费用估算**：
- 消防改造/验收：¥20,000-50,000
- 装修符合要求：¥50,000-150,000
- 代办服务费（可选）：¥10,000-30,000

### 三、互联网资质（线上业务）

#### ICP 备案（必须）

所有网站/App都需要 ICP 备案：

| 项目 | 说明 |
|------|------|
| 办理部门 | 通过服务器提供商代办（阿里云、腾讯云等） |
| 所需材料 | 营业执照、法人身份证、域名证书、网站信息 |
| 办理周期 | 约 20 个工作日 |
| 费用 | 免费（云服务商代办） |

#### ICP 许可证（经营性网站）

如果网站/App 有收费业务，需要 ICP 许可证：

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   ICP 备案（非经营性）              ICP 许可证（经营性）      │
│   ─────────────────                ──────────────────       │
│   • 免费信息展示                    • 收费服务              │
│   • 自有课程销售                    • 平台抽佣              │
│   • 免费                           • 广告收入               │
│   • 约 20 个工作日                  • 需要公司资质审核       │
│                                    • 约 40-60 个工作日      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**ICP 许可证申请条件**：
- 公司注册资本 ≥ 100万元（认缴即可）
- 有必要的技术人员
- 有可行的发展规划

#### 公安备案

ICP 备案通过后 30 天内，还需要在"全国互联网安全管理服务平台"做公安备案。

| 项目 | 说明 |
|------|------|
| 网址 | www.beian.gov.cn |
| 周期 | 3-10 个工作日 |
| 费用 | 免费 |

### 四、进校资质（B2B/B2G）

#### 参与学校课后服务要求

要进入学校"白名单"，需满足：

1. ✅ 持有有效期内的办学许可证
2. ✅ 具有独立法人资格
3. ✅ 持有营业执照或民办非企业单位登记证
4. ✅ 入选当地非学科类培训机构"白名单"
5. ✅ 无违规办学记录
6. ✅ 无安全事故记录
7. ✅ 银行资金监管账户（部分地区要求）

#### 白名单申请流程

各地流程不同，一般：
1. 在教育局官网或政务平台提交申请
2. 提供办学许可证、师资资料、课程方案
3. 教育局审核并公示
4. 进入白名单后可参与学校课后服务采购

## 运营合规

### 收费规定

| 规定 | 要求 |
|------|------|
| 单次收费上限 | 不超过 **5000 元** |
| 收费周期 | 不超过 **3 个月**（或60课时） |
| 预收费监管 | 部分地区要求资金监管账户 |
| 退费机制 | 需公示退费规则 |

**重要**：这是硬性规定，违反会被处罚。建议：
- 设计课程包时考虑这个限制
- 可以分期收费，但每期不超过3个月

### 师资要求

| 要求 | 说明 |
|------|------|
| 无犯罪记录 | 依据《未成年人保护法》第62条 |
| 健康证明 | 无传染病、心理疾病 |
| 资质证书 | 教师资格证/专业技能证（推荐但非强制） |
| 实名公示 | 教师信息需在机构公示 |

### 广告宣传

| 禁止行为 | 说明 |
|---------|------|
| ❌ 虚假宣传 | 夸大教学效果 |
| ❌ 制造焦虑 | "不学编程就落后" |
| ❌ 升学承诺 | 保证竞赛获奖/升学加分 |
| ❌ 师资造假 | 虚构教师背景 |

## 业务模式选择

### 模式对比

| 模式 | 资质要求 | 投入 | 风险 | 适合谁 |
|------|---------|------|------|--------|
| **纯SaaS平台** | 公司+ICP备案 | 低 | 低 | 技术团队 |
| **线上培训** | 公司+ICP+（部分地区需办学许可） | 中 | 中 | 有教研能力 |
| **线下培训** | 公司+办学许可证 | 高 | 高 | 有资金和运营能力 |
| **B端进校** | 公司+办学许可证+白名单 | 高 | 中 | 有政府资源 |

## 关键成功因素

### 必须做到

1. **合规运营** - 资质齐全，严守政策红线
2. **差异化产品** - 不要跟头部正面竞争
3. **优质服务** - 中国家长重视服务体验
4. **控制成本** - 获客成本高，注意现金流
